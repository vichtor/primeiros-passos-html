1- ENTENDENDO A COMUNICAÇÃO CLIENT X SERVER

HISTÓRIA DA WEB

A história da Web começa inicialmente com os primeiros computadores que surgiram como máquinas de cálculo lógico, devido às necessidades de comunicação durante a Segunda Guerra Mundial.

A Primeira Geração de computadores se inicia “teoricamente” em 1944, quando foi criado o Mark I pelos Estados Unidos. Um computador eletromecânico que realizava diversos cálculos, porém com funcionamento em válvulas possuía um tamanho gigantesco e necessitava de um ambiente bem amplo para ser alocado.

Em 1946 os Alemães criaram o Colossus, com intenção de realizar cálculos que ajudariam na Segunda Guerra Mundial. Em contrapartida também em 1946, os Americanos lançaram o ENIAC, o primeiro computador eletrônico com funcionamento baseado em válvulas e relés. Por conta disso a “máquina” esquentava muito e atraía insetos que causavam curtos no equipamento, surgindo assim o termo “Bug” (supostamente criado por Grace Murray Hopper).

A Segunda Geração se passou entre os anos de 1959 e 1965, onde as Válvulas foram substituídas pelos Transistores. Essa troca de componentes permitiu que os computadores diminuíssem bastante o seu tamanho físico, podendo assim serem mudados de local com mais facilidade.

Já entre os anos de 1965 e 1970 tivemos a chegada da Terceira Geração, com a criação dos Circuitos Integrados (transistores, capacitores, resistores). Dessa maneira, a diminuição de tamanho físico e aumento de capacidade lógica estavam apenas começando.

Chegando enfim a Quarta Geração que se inicia em 1971 e se estende até os dias atuais, temos além dos Circuitos Integrados, a criação de Microprocessadores e outros tipos de chips e componentes que vem evoluindo através da criação contínua de novos aparelhos eletrônicos e tecnológicos, rumo a Quinta Geração.

Com toda essa evolução dos computadores, tivemos também a invenção da Internet que teve início lá atrás ainda no século XIX, com as primeiras formas de comunicação a distância... Como o Telégrafo (Código Morse) por exemplo.

Em 1958, foi esticado pela primeira vez um cabo submarino entre dois Continentes, estabelecendo uma comunicação em longa distância via cabo físico. Isso devido a Guerra Fria e a Corrida Espacial entre EUA e a antiga URSS na década de 1950. Foi definitivamente a década em que briga pelo poder e sempre chegar à frente do inimigo, decretaram uma grande corrida pela melhor tecnologia.

Para facilitar a troca de informações e temendo ataques soviéticos, o Departamento de Defesa dos Estados Unidos criou a Arpanet, a primeira rede de internet. Em 29 de outubro de 1969, estabeleceu-se a primeira conexão entre a Universidade da Califórnia e o Instituto de Pesquisa de Stanford, marcando o envio do primeiro e-mail. Essa primeira troca de informação foi através de apenas uma palavra, LOGIN... Chegando apenas o “LO”, ou seja, a primeira troca de informação e a primeira falha de comunicação.

Em 1989, o cientista britânico Tim Berners-Lee desenvolveu a www, a Rede Mundial de Computadores. Embora não tenha inventado a conexão entre computadores geograficamente distantes (como a Arpanet), Berners-Lee criou o modelo que usamos diariamente para acessar a internet. Essa década ficou conhecida como o “boom da internet”, quando ela se popularizou globalmente. Surgiram diversos navegadores, como o Mosaic, Internet Explorer, Netscape, entre outros, e a internet se tornou uma rede global de computadores conectados. A Internet ultrapassou barreiras, aproximando pessoas, culturas, mundos e informações. Hoje, a Internet é essencial para trabalho, pesquisa, diversão, comunicação, educação, informação e até compras de produtos diversos.


QUAL A DIFERENÇA ENTRE CLIENT E SERVER
"A estrutura cliente-servidor é um modelo de arquitetura de rede amplamente utilizado em ambientes de TI. Nesta abordagem, os computadores são divididos basicamente em dois grupos: servidores, que fornecem serviços ou recursos, e clientes, que solicitam estes serviços e recursos, recebendo-os como respostas".

O CLIENT (ou CLIENTE) é um computador ou qualquer outro dispositivo configurado para utilizar os serviços oferecidos pelo SERVER (ou SERVIDOR). Ele faz solicitações ao servidor para obter informações ou executar tarefas. O CLIENT não compartilha seus recursos com outros computadores, a não ser que ele permita. Por exemplo, quando acessamos uma página da web, o computador (CLIENT) solicita o conteúdo a partir de um servidor web (SERVER).

O SERVER, é nada mais nada menos que um computador configurado em uma rede para oferecer serviços. Ele processa solicitações e fornece recursos ou conteúdo a outros computadores (CLIENTS). Alguns exemplos de servidores seriam servidores web (que hospedam sites), servidores de e-mail, servidores de arquivos e servidores de banco de dados. O servidor compartilha seus recursos (como processamento, armazenamento e largura de banda) com os clientes.

Em resumo, o SERVER é o provedor de serviços, enquanto o CLIENT é aquele que consome esses serviços. Essa comunicação é conhecida como CLIENT-SERVER e é essencial para o funcionamento da INTERNET e de muitas aplicações em redes de computadores.


O QUE É LINGUAGEM DE PROGRAMAÇÃO

Linguagem de programação é um método padronizado para expressar instruções para um computador. Ela consiste em um conjunto de regras sintáticas e semânticas usadas para definir um programa de computador.

Por sua Definição, linguagem de programação é uma linguagem formal que permite que um programador escreva um conjunto de ordens, ações consecutivas, dados e algoritmos. Essas instruções são interpretadas pela máquina, controlando seu comportamento físico e lógico.

Sendo assim, podemos considerar que a Linguagem de Programação é um sistema de comunicação estruturado, composto por símbolos, palavras-chave e regras que permitem a interação entre programador e máquina. Portanto, são ferramentas essenciais para desenvolver aplicativos digitais e possibilitam a comunicação entre humanos e máquinas.

O QUE SÃO TAGS HTML E SEUS ATRIBUTOS

As Tags são elementos que definem a estrutura e o significado do conteúdo em um documento HTML. Elas são utilizadas para formatar um texto, inserir imagens ou links e muitos outros recursos. Sua função é indicar ao navegador como interpretar o conteúdo de uma página da web.

São encobertas por uma “abertura” e um “fechamento” que a delimitam, com o símbolo “menor que” ( < ) para abertura e “maior que, com barra” ( /> ) para fechamento, definindo da seguinte maneira a estrutura que será utilizada: < tag > conteúdo < /tag >. Mas também existem tags que não precisam de fechamento para sua execução e são definidas da seguinte maneira: < tag />.

Já os Atributos, são usados para personalizar as Tags, modificando sua estrutura ou funcionalidade através de comandos ou informações específicas dadas a uma Tag. Os Atributos são usados para que a Tag faça tal comando, porém com algum detalhe ou variação específica em relação a Tag original. Sendo assim, eles são utilizados da seguinte forma: < tag atributo=”valor” > conteúdo < /tag >.

Para finalizar, citarei 5 exemplos de Tags abaixo:

  1. TEXTOS - Tag "p" entre os sinais < >, fechando com "/p";
  2. LISTA ORDENADA - Tag "ol" entre os sinais < >, fechando com "/ol";
  3. LISTA NÃO ORDENADA - Tag "ul" entre os sinais < >, fechando com "/ul";
  4. ITENS DA LISTA - Tag "li" entre os sinais < >, fechando com "/li";
  5. LINKS - Tag "a + atributo = "valor" " entre os sinais < >, fechando com "/a".
